## Usage:

-   Prerequisites: `node` and `npm`

#### Clone the repo:

```bash
git clone https://github.com/levsthings/clientside-toolchain
rm -rf .git && git init
```

#### Install dependencies:

```bash
npm i
```

#### Development mode with hot reloading:

```
npm run dev
```

#### Bundling in production mode:

```
npm run build
```
